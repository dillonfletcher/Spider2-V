We need to get this code running smoothly. 
1. Click "Restart the kernel and run all cells"
2. Click "Restart"
3. Then, we found the error ModuleNotFoundError: No module named 'nltk'. we need to add this package in terminal
4. Switch to "terminal"
5. Input the code  ```
    pip install nltk
   ```
6. Swith Back to Chrome, and click "Restart the kernel and run all cells"
7. In the pop-up window, click "Restart"
8. We found new error,  
  ```
LookupError: 
**********************************************************************
  Resource stopwords not found.
  Please use the NLTK Downloader to obtain the resource:

  >>> import nltk
  >>> nltk.download('stopwords')
  
  For more information see: https://www.nltk.org/data.html

  Attempted to load corpora/stopwords

  Searched in:
    - '/home/user/nltk_data'
    - '/home/user/anaconda3/envs/jupyter/nltk_data'
    - '/home/user/anaconda3/envs/jupyter/share/nltk_data'
    - '/home/user/anaconda3/envs/jupyter/lib/nltk_data'
    - '/usr/share/nltk_data'
    - '/usr/local/share/nltk_data'
    - '/usr/lib/nltk_data'
    - '/usr/local/lib/nltk_data'
**********************************************************************
```
9. Switch back to "Terminal"
10. Run the bash code
    ```
    python -c "import nltk; nltk.download('stopwords')"
    ```
11. Switch back to Chrome
12. Click "Restart the kernel and run all cells"
13. Click "Restart"
14. The code execute successfully
15. Click the "Save and create checkpoint" to save the result.
