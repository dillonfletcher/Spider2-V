This job is to connect Hasura to the existing snowflake database and write GraphQL queries on Hasura. In the opened Snowsight web page,
1. Click the blue button on the middle of the panel named "Launch Button".
2. On the new web page, click the "DATA" button on the top of the page.
3. On the Data Manager page, if there is any database related to videos, click the red button named "Remove" on the right. If not, skip this step.
4. In the pop-up window, type in {database_name} to confirm the remove action. For example, if the database name is videos, type in videos. If you skip the previous step, skip this step too.
5. On the right-bottom of this pop-up window, click the red button "Remove".
6. On the bottom of the window, select "Snowflake".
7. On the right-bottom of the window, click the "Connect Existing Database" button.
8. Type in "videos" to the input field "Database Name".
9. Open the information.txt file on the desktop of virtual machine.
10. Replace the placeholders in the JDBC format with values provided by information.txt to get the JDBC string "jdbc:snowflake://HTEVMCF-AZ36651.snowflakecomputing.com/?user=XLANG&password=Spider2.0&warehouse=compute_wh&db=videos&schema=public"
11. Return to the Chromium.
12. Fill in the input field "JDBC Connection URL" with the JDBC string you got previously.
13. Click "Connec Database" button on the right-bottom.
14. On the left panel, Click "videos" button under "Databases".
15. Select all check boxes on the middle of the window.
16. Click the yellow button "Track Selected".
17. Click the "API" button on the top of the page.
18. In the input field of "GraphQL", type in the following code in the input box with the "# Try out GraphQL queries here" prompt.
```
query AllCustomers {
  SUBSCRIPTIONS(offset: 50, where: {WAS_SUBSCRIPTION_PAID: {_eq: "Yes"}}, limit: 100) {
    CUSTOMER_ID
    CREATED_DATE
  }
}
```
19. Click the "Execute Query" button(With triangle symbol) just above the input box.
20. Open Terminal on your virtual machine.
21. Run the following commands.
```
cd Desktop
code result.txt
```
22. Copy the query results on the right of the Chromium window to the result.txt.
23. Save result.txt
