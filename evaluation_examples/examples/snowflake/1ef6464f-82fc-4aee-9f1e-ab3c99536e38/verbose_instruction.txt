This task is to create a custom filter to filter the query results. In the opened Snowsight web page,
1. Click "Projects >> Worksheets" on the left menu panel.
2. Click "+" button on the right;
3. In the drop-down menu, select "SQL Worksheet".
4. On the top-left of the worksheet, click the filter button which has an icon with two horizontal lines.
5. On the drop-down menu, click "Manage Filters".
6. On the top-right of the pop-up window, click "+ Filter" button.
7. Type in "Title_Keyword" to the input field "Display Name".
8. Type in "title_keyword" to the input field "SQL Keyword".
9. Type in "Return unique job titles." to the input field "Description".
10. Select "COMPUTE_WH" for the "Warehouse".
11. Click the blue button "Write Query" below.
12. In the pop-up window, Select "DB >> PUBLIC".
13. Copy the code below to the filter query edit field.
```
SELECT distinct job_title from salary;
```
14. Click "Run" button on the top-right. There will be several job titles on the result field.
15. Click "Done" button on the top-left.
16. Click "Save" button on the bottom-right.
17. Click "Done" button on the bottom-right.
18. Copy the query from query.txt on the desktop to the worksheet. The contents of query.txt are also given below.
```
SELECT avg(salary) as average_salary FROM salary where job_title = :title_keyword;
```
19. Click "Title_Keyword" button on the top of the worksheet.
20. In the drop-down menu, Select the second option "Data Analyst".
21. Click "No Database selected" button on the top of the worksheet, then select "DB >> PUBLIC".
22. Click "Apply" button just to the right of the "Title_Keyword" button.
23. Click the "Download Results" button on the top-right of the result panel.
24. In the drop-down menu, click "Download as .csv".
25. In your virtual machine, navigate to the /home/user/Download directory.
26. Rename the downloaded file to "result.csv".
