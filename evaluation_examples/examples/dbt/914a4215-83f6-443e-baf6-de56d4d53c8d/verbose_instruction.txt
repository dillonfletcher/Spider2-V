This tasks requires you to set the execution commands of a job.
1. Click "Deploy" in the upper panel.
2. Choose "Jobs".
3. Click "Create job", and choose "Deploy job".
4. In job settings section, set the job name to "init".
5. In execution settings section, check "Generate docs on run" tick box.
6. In execution settings section, set the three commands in the following order:
* dbt seed
* dbt test
* dbt run
7. Click "Save".
